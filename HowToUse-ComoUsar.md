## How to Use / Como Usar

### English

1. Open the app.
2. Create a class in the **Classes** section.
3. Add students to the selected class.
4. Register activities, exams/AVs, and bonus points.
5. Use the roulette tool for classroom dynamics.
6. Edit or remove students and classes when needed.
7. Confirm deletion before removing important data.
8. Import data from **Excel** or **JSON** files.
9. Export your data to keep a backup or move it to another device.
10. All data is stored locally and can be used offline.

### Português

1. Abra o aplicativo.
2. Crie uma turma na seção **Turmas**.
3. Adicione alunos à turma selecionada.
4. Registre atividades, avaliações/AVs e pontos extras.
5. Use a ferramenta de roleta para dinâmicas em sala.
6. Edite ou remova alunos e turmas quando necessário.
7. Confirme a exclusão antes de remover dados importantes.
8. Importe dados por arquivos **Excel** ou **JSON**.
9. Exporte seus dados para manter backup ou levar para outro dispositivo.
10. Todos os dados ficam salvos localmente e podem ser usados offline.
