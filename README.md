# School Grades Offline App / Aplicativo Offline de Notas Escolares

## English

### Overview
This project is an offline educational application developed to help teachers manage school classes, students, grades, activities, bonus points, and a classroom roulette tool in a practical and organized way.

The app was designed to work locally, allowing data storage directly on the device, without requiring a constant internet connection.

### Main Features
- Class management
  - create classes
  - rename classes
  - delete classes with confirmation
- Student management
  - add students
  - edit student data
  - remove students with confirmation
- Grade organization
  - activities
  - exams / AVs
  - bonus points
- Classroom roulette tool
- Local offline storage
- Import and export support
  - Excel
  - JSON
- Automatic class creation when importing data with new class names

### Purpose
This app was created for educational use, especially to support teachers in organizing classroom information in a simple, visual, and offline-friendly way.

### Technologies
- Flutter
- Local storage
- Excel and JSON file import/export

### Important Note
This project was developed with the help of Artificial Intelligence as a support tool for planning, coding assistance, and refinement of features.

## Português

### Visão geral
Este projeto é um aplicativo educacional offline desenvolvido para ajudar professores no gerenciamento de turmas, alunos, notas, atividades, pontos extras e uma ferramenta de roleta para uso em sala de aula de forma prática e organizada.

O aplicativo foi pensado para funcionar localmente, permitindo o armazenamento dos dados diretamente no dispositivo, sem necessidade de conexão constante com a internet.

### Principais funcionalidades
- Gerenciamento de turmas
  - criar turmas
  - renomear turmas
  - excluir turmas com confirmação
- Gerenciamento de alunos
  - adicionar alunos
  - editar dados dos alunos
  - remover alunos com confirmação
- Organização de notas
  - atividades
  - avaliações / AVs
  - pontos extras
- Ferramenta de roleta para sala de aula
- Armazenamento local offline
- Suporte para importação e exportação
  - Excel
  - JSON
- Criação automática de novas turmas ao importar dados com nomes de turmas diferentes

### Objetivo
Este aplicativo foi criado para uso educacional, especialmente para auxiliar professores na organização das informações da sala de aula de forma simples, visual e funcional mesmo sem internet.

### Tecnologias
- Flutter
- Armazenamento local
- Importação e exportação em Excel e JSON

### Observação importante
Este projeto foi desenvolvido com ajuda de Inteligência Artificial como ferramenta de apoio no planejamento, assistência de código e refinamento de funcionalidades.
